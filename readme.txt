Ce repo contient un projet de création de site web front-end pour le module 307 de l'EMF.


Le dossier documentation contient mon RP qui lui même contient la documentation de projet.
Le dossier code contient tout le code de ce projet.




Le but de ce projet est d'utiliser différentes APIs, en l'occurence le mien aura pour thème le fitness
et aura une page d'accueil, une page regroupant un grand nombre d'exercices et des infos sur ces derniers
dans une autre page, une page permettant de rechercher des activités et le nombre de calories brûlées par celles-ci
et pour finir une dernière page qui affiche une carte des centres de fitness du canton de Fribourg.
